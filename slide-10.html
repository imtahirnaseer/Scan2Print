Core Protocol

Secure Purge & Wipe

No Disk Writes

Files are streamed directly to RAM (Redis) and decrypted only in the Print Agent's volatile memory. No temp files are ever created on the HDD.

Anti-Residue

Unlike WhatsApp/Email, the "Sent" folder doesn't exist. Once the print command executes, the file binary is overwritten and the link expires.
